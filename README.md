# Weather-App
This Weather Web App is a simple and stylish tool that provides real-time weather updates for any city in the world. Developed using HTML, CSS, and JavaScript, this app fetches weather data from the OpenWeather API and displays information such as:

Current Temperature
Humidity
Wind Speed
Weather Conditions (Rain, Clear, Cloudy, etc.)
The app allows users to enter a city name, view the current weather, and receive helpful weather icons based on the conditions. Whether you're checking the forecast for your next trip or simply staying updated on the weather, this app has you covered!

Features:
Search by City: Enter any city name to get real-time weather data.
User-friendly Interface: A clean and modern design with intuitive navigation.
Weather Icons: Displays various weather icons (rain, snow, clear sky, etc.) based on the current conditions.
Responsive Layout: Works smoothly across devices with different screen sizes.
Technologies Used:
HTML for structuring the content.
CSS for styling and responsive design.
JavaScript to handle weather data fetching and user interactions.
OpenWeather API to fetch the latest weather data.
This project was a great way to practice working with APIs, DOM manipulation, and front-end design. It's simple, yet a fun way to learn how to connect with external data and display it effectively.
